📌 About This Folder

This directory contains a collection of basic C programs that focus on fundamental programming concepts.
It is part of my journey to master C programming from scratch and build a solid base for advanced topics.

⸻

📂 Programs Included

🔢 Prime Number Checker

File: Primechecker.c
✔ Checks whether a number is prime
✔ Handles edge cases (numbers ≤ 1)
✔ Uses loops and conditional logic

⸻

📄 Basic Starter Program

File: Prog1.c
✔ Demonstrates the structure of a C program
✔ Covers input and output basics

⸻

🧮 Calculator

File: calculator.c
✔ Performs:

* Addition
* Subtraction
* Multiplication
* Division

✔ Uses switch-case
✔ User input driven

⸻

🔁 Factorial Calculator

File: factorial.c
✔ Calculates factorial using long long
✔ Handles:

* Negative numbers (invalid case)
* 0! = 1
    ✔ Works for larger values (within limits)

⸻

📈 Fibonacci Series

File: fibonacci.c
✔ Generates Fibonacci sequence
✔ Uses iteration and sequence logic

⸻

⚙️ How to Run

# Compile
gcc filename.c -o output
# Run
./output

📌 Example:

gcc calculator.c -o calc
./calc
